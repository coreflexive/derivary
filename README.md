# Derivary

Structured mathematical derivations in relation algebra, predicate calculus, Galois connections, and calculational proof.

This repository is intended to be opened as an Obsidian vault.

Each directory gathers the Typst source and rendered SVG artefacts for a particular theory, predicate, construction, or calculation. The folder structure is part of the presentation: theories contain observations, and observations contain structured derivations.

To browse the derivations directly, open the generated SVG files in each `Observation/` directory.

Obsidian provides the browsing layer: folders, links, backlinks, tags, and rendered SVGs make the archive directly explorable.

Typst provides the source language and rendering pipeline for the mathematical artefacts. Shared Typst helpers are in `lib/`.

I may add commentary where it is useful, but the calculations are intended to stand on their own.

This is a working archive.
